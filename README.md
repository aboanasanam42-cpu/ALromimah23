# CloudWorker AI

Architecture Specification & Codebase Initialization for "CloudWorker AI" Application (Android 15).

## Project Core Configuration
* **Package Name:** `ai.albader.alromimh.com`
* **Target SDK:** Android 15 (SDK 35)
* **UI Framework:** Jetpack Compose
* **Architecture:** Clean Architecture

## Core Features
* **Dual-Engine Modes:** Local App Mode & Cloud Mode.
* **100-Point Scoring Algorithm:** Evaluating remote work opportunities.
* **Security:** AES encryption & Biometric/PIN protection.

## CI/CD
* **CircleCI:** Automated build pipeline.
* **Codemagic:** Signed release APK generation and publishing.
