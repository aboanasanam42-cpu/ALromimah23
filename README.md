# CloudWorker AI

CloudWorker AI is an Android application for discovering, saving, and applying to job opportunities with AI-powered scoring, biometric security, Material You theming, and Arabic/English support.

## Features
- AI Opportunity Scoring
- Room Database with offline-first caching
- Hilt Dependency Injection
- Material You Dynamic Theming
- Biometric Authentication
- Arabic & English Localization
- Jetpack Compose UI

## Tech Stack
- Kotlin 1.9.20
- Jetpack Compose
- Room
- Hilt
- Retrofit + OkHttp
- Navigation Component
- Material3

## Build
```bash
./gradlew assembleRelease
```
